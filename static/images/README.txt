Images du site
==============

Photo de profil (page d'accueil) :
  Mettez un fichier nommé exactement "profil.jpg" (ou profil.png) dans ce dossier.
  Exemple : static/images/profil.jpg

La galerie d'images charge les photos depuis les dossiers de votre ordinateur
(scan du disque), pas depuis ce dossier. Ce dossier sert uniquement à la photo
de profil affichée sur la page d'accueil.
